# SIT305 Task 6.1D - AI Learning Assistant App

## Overview
This project is an Android learning assistant app developed for SIT305 Task 6.1D. It is designed to support students with revision, guided practice, and topic-based study activities through a simple and interactive mobile interface.

## Features
- Login and registration
- Interest selection
- Generated study tasks
- Answer feedback
- Create 3 flashcards from a topic
- Results screen with prompt and response display
- Loading and error handling

## App Flow
The app follows this learning flow:
1. Login or register
2. Select study interests
3. View generated tasks on the home screen
4. Open a task and complete study activities
5. Review the returned result on the results screen

## AI Support
This app includes built-in AI study support for learning activities. It can explain answers and generate flashcards based on a selected topic. The app also displays the prompt and returned response in the interface.

## Tools Used
- Kotlin
- Android Studio
- Jetpack Compose
- Navigation Compose
- Material 3
- Retrofit
- ViewModel

## Running the Project
1. Open the project in Android Studio
2. Sync Gradle
3. Run the app on an emulator or Android device

## Notes
This project uses sample data to simulate user information, study interests, and generated learning tasks.
